Java + POO + MySQL

Gestión de Usuarios

Sistema de administración CRUD desarrollado en Java para gestionar registros de forma eficiente mediante una interfaz intuitiva.

Permite crear, leer, actualizar y eliminar datos, además de buscar y modificar perfiles usando identificadores únicos, con almacenamiento de usuarios en una base de datos MySQL.

Logro técnico: integración con MySQL para guardar usuarios, consultar registros y validar campos, asegurando consistencia en la información.

Tecnologías: Java, MySQL y Programación Orientada a Objetos.
